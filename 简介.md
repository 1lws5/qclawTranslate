# qclawTranslate 版本简介

## v1.4.0 (2026-07-08)
- 翻译：OpenAI 兼容 API（DashScope 等），支持中英日韩等多语言自动检测
- TTS：CosyVoice WebSocket 流式合成，~1.5 秒出音
- 热键：Ctrl+Q 划词翻译 / Ctrl+E 划词朗读
- 划词：UIA 检测选中 + Ctrl+C 回退搬运
- 托盘：Windows 系统托盘常驻，双击恢复
- UI：暗黑主题，QSS 圆角边框
- 打包：PyInstaller exe（37MB）

已知问题：
- 热键捕获正常，但触发后偶尔拿不到选中文字（UIA/Ctrl+C 链路待修复）
- 打包后首次启动热键可能被旧实例占用（已加单实例互斥锁）
