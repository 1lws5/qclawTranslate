## 五、核心技术栈

| 层 | 实际方案 | 状态 |
|---|---|---|

| 选中读取路径1 | UIA `GetFocusedElement()` → TextPattern 偏移截取 | ✅ 已实现 |
| 选中读取路径2 | UIA `ElementFromHandle(hwnd)` → TextPattern 偏移截取 | ✅ 已实现 |

| SendInput 结构体 | INPUT 联合体含 MOUSEINPUT+KEYBDINPUT+HARDWAREINPUT，sizeof=40 | ✅ 已修复 |
| config 路径 | `sys.frozen` 时用 `sys.executable` 定位，否则用 `__file__` | ✅ 已实现 |
| config 结构 | 嵌套结构(translate/tts/pronunciation/subtitle/general) + 旧格式自动迁移 | ✅ 已实现 |
| 翻译 | DashScope Compatible API (`qwen-turbo`) + TCP连接复用 + LLM streaming | ✅ |
| TTS | CosyVoice WebSocket `cosyvoice-v3-flash` / `longanyang` | ✅ |
| 播放 | QMediaPlayer + QMediaContent，本地文件播放 | ✅ |
| 音频缓存 | `_cache_path()` 基于 text+voice+model 的 MD5 生成路径 | ✅ |
| UI | PyQt5 + QSS 暗黑主题(边框1px/圆角8px/对比度提升) | ✅ |
| 设置对话框 | 三层结构：外层 SettingsDialog → EngineSettingsDialog / GeneralSettingsDialog | ✅ |
| 托盘 | QSystemTrayIcon + QMenu | ✅ |
| 单实例 | `CreateMutexW` + `GetLastError` | ✅ |
